# ATOC Website Phase A

Local review build for a replacement ATOC website.

This project is not deployed and does not modify the live WordPress site at `www.atocbar.com`.

## Local Commands

```powershell
npm install
npm run dev
npm run build
```

The dev server is configured for local review at `http://127.0.0.1:8789`.

## Safety

- No WordPress login.
- No publishing or deployment.
- No DNS or hosting changes.
- No live API keys.
- No Seedance, Ark, Comfy, or AI generation calls.
- AI-Ops-Hub source assets were copied read-only into this project.

## Content Notes

Legacy site facts are recorded in `docs/legacy-site-audit.md`. Current address, hours, contact, promotions, and menu details must be confirmed before publication.
